# 🎓 IT Student Assistant Bot

Telegram-бот для студентов IT-специальностей. Помогает с подготовкой к собеседованиям, учебными проектами и повторением материала.

## ✨ Возможности

### 📝 Дневник ошибок
- Сохранение ошибок с OCR распознаванием скриншотов
- Добавление решений
- Интервальные повторения для запоминания

### 🎴 Карточки (Flashcards)
- Создание карточек с вопросами-ответами
- Алгоритм интервальных повторений (SM-2)
- Статистика правильных ответов

### 🍅 Pomodoro таймер
- Фокус-сессии 25/5 минут
- Блокировка отвлекающих сообщений
- Интеграция с GitHub
- Статистика по темам

### 💡 Генератор идей проектов
- Случайные идеи pet-проектов
- Добавление своих идей
- Фильтрация по сложности

## 🛠 Технологии

- Python 3.12
- aiogram 3.x
- MySQL / aiomysql
- Tesseract OCR
- Pillow
- python-dotenv
